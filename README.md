# Duplicate

Duplicate exturder settings for Cura in case of multi-extruder machine. Copy parameters that are different in :
- resolution
- shell (up to Cura 4.8)
- walls (from Cura 4.9)
- top/bottom (from Cura 4.9)
- infill
- material
- speed
- travel
- cooling
- dual
- experimental
- blackmagic
- meshfix

From one extruder to the others.
